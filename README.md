# Sadpara Shilajeet - E-commerce Frontend

A modern, premium e-commerce frontend for selling pure Himalayan Shilajit.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Redux Toolkit
- React Hook Form + Zod
- Framer Motion
- React Helmet Async

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
